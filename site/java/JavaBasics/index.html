    ‚úÖ 1. Java Basics (Explained Simply)

    üî∏ What is Java? (Features)

    Java is a programming language used to build applications (websites, mobile apps, games, etc.).
    It's also a platform because it runs on the Java Virtual Machine (JVM).

    Key Features: - üß† Simple ‚Äì Easy to learn and use. - üîÑ Object-Oriented ‚Äì Uses objects and classes to organize code. - üíª Platform Independent ‚Äì Runs on any device with a JVM (Write Once, Run Anywhere). - üõ°Ô∏è Secure ‚Äì Protects against viruses and memory access issues. - ‚öôÔ∏è Robust ‚Äì Handles errors and manages memory well. - üîÄ Multithreaded ‚Äì Can perform multiple tasks at once.

    üî∏ Installing JDK, JVM, JRE

    • JDK (Java Development Kit): Tools needed to write and run Java programs (includes compiler, debugger, etc.).
    • JRE (Java Runtime Environment): Allows you to run Java programs (includes JVM + libraries).
    • JVM (Java Virtual Machine): Executes Java bytecode on your computer.

    ‚úÖ To code in Java, install the JDK (which includes both JRE and JVM).

    üî∏ Java Program Structure

    A typical Java program looks like this:

    public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, world!");
    }
}
    • class HelloWorld ‚Üí Every program is inside a class.
    • main() method ‚Üí Entry point; program starts here.
    • System.out.println() ‚Üí Prints to the console.

    üî∏ Writing Your First Program (HelloWorld)

    Steps:

    1. Open a text editor or IDE (like VS Code or IntelliJ).
    2. Write the HelloWorld code shown above.
    3. Save the file as HelloWorld.java.

    üî∏ Compilation and Execution Flow

    ‚úÖ Compile:
    Run the following command: 

    javac HelloWorld.java

    ‚û°Ô∏è Converts source code into bytecode (HelloWorld.class)

    ‚ñ∂Ô∏è Run: Run the following command:

    java HelloWorld

    ‚û°Ô∏è JVM executes the program and prints:

    Hello, world!
    Step Output
    Source Code .java file
    Compilation .class bytecode
    Execution (JVM) Program runs

    ‚úÖ 2. Data Types and Variables (Explained Simply)

    üî∏ Primitive Data Types (int, float, char, etc.)

    These are basic types built into Java.
    They store single values and are not objects.

    Type Example Description
    int int age = 25; Stores whole numbers
    float float pi = 3.14f; Stores decimal numbers (use f)
    double double d = 10.56; Stores large decimal numbers
    char char grade = 'A'; Stores a single character
    boolean boolean isOn = true; Stores true or false
    byte byte b = 100; Very small integer (1 byte)
    short short s = 20000; Small integer (2 bytes)
    long long l = 100000L; Large integer (add L at the end)

    üî∏ Type Casting (Implicit and Explicit)

    Type Casting means converting one data type into another.

    ‚úÖ Implicit Casting (Widening Conversion)
    Automatically converts a smaller type to a larger type. No data is lost.

    int a = 10;
double b = a;  // int is automatically converted to double

    ‚úÖ Explicit Casting (Narrowing Conversion)
    Manually converts a larger type to a smaller type. May lose data.

    double x = 10.5;
int y = (int) x; // double is cast to int, decimal part is lost

    üîí final ‚Äì Used to create constants. Once assigned, the value cannot be changed.

    ‚úÖ Example:

    final int MAX_USERS = 100;// MAX_USERS = 200; // ‚ùå Error: cannot assign a value to final variable

    üåê static ‚Äì Belongs to the class, not to individual objects.
    It is shared among all instances of the class.

    ‚úÖ Example:

    class Counter {
    static int count = 0;  // static variable shared by all instances

    Counter() {
        count++;           // increment count whenever an object is created
    }
}

public class Main {
    public static void main(String[] args) {
        Counter c1 = new Counter();
        Counter c2 = new Counter();
        Counter c3 = new Counter();

        System.out.println(Counter.count); // Output: 3
    }
}

    üìù Explanation:

    • The count variable is marked as static, so it's not tied to a specific object.
    • No matter how many objects (c1, c2, c3) are created, they share the same static variable.
    • This is why Counter.count gives 3 ‚Äî the total number of objects created.

    ‚ú® var (Java 10+) ‚Äì Automatically detects the data type based on the assigned value.

    ‚úÖ Example:

    var name = "Alice";   // treated as String
var age = 30;         // treated as int
var price = 99.99;    // treated as double

    ‚úÖNote: var can only be used inside methods, not for class-level variables or fields. ‚ùåInvalid Usage:

    class Test {
    var a = 10;  // ‚ùå Error: 'var' is not allowed at class level
}

    ‚úÖValid Usage:

    public class Main {
    public static void main(String[] args) {
        var message = "Hello Java";  // ‚úÖ OK inside method
    }
}

    ‚úÖ 3. Operators (Explained Simply)

    Java operators are special symbols that perform operations on variables and values.

    üî∏ Arithmetic Operators

    Used for basic math operations.

    Operator Description Example Result
    + Addition 5 + 3 8
    - Subtraction 5 - 3 2
    * Multiplication 5 * 3 15
    / Division 10 / 2 5
    % Modulus (remainder) 10 % 3 1

    üî∏ Relational (Comparison) Operators

    Used to compare two values. Returns true or false.

    Operator Description Example Result
    == Equal to 5 == 5 true
    != Not equal to 5 != 3 true
    > Greater than 5 > 3 true
    < Less than 5 < 3 false
    >= Greater than or equal 5 >= 5 true
    <= Less than or equal 4 <= 3 false

    üî∏ Logical Operators

    Used to combine multiple boolean expressions.

    Operator Description Example Result
    && Logical AND true && false false
    || Logical OR true || false true
    ! Logical NOT !true false

    üî∏ Assignment and Unary Operators

    Assignment Operators: Used to assign values to variables.

    int x = 10;
x += 5;  // same as x = x + 5;
    Operator Meaning
    = Assign
    += Add and assign
    -= Subtract and assign
    *= Multiply and assign
    /= Divide and assign
    %= Modulus and assign
    ---
    ### üî∏ Unary Operators

    Used with a single operand to perform operations like increment, decrement, or negation.

    Operator Description Example Result
    + Unary plus +x +10
    - Unary minus -x -10
    ++ Increment x++ x + 1
    -- Decrement x-- x - 1
    ! Logical NOT !true false

    üî∏ Ternary Operator

    Used as a shorthand for if-else statements.

    ‚úÖ Example:

    int age = 18;
String result = (age >= 18) ? "Adult" : "Minor";

    ‚úÖ If the condition is true, it returns "Adult", else "Minor".

    üìå Syntax:

    (condition) ? value_if_true : value_if_false;

    üî∏ Bitwise Operators

    Used for operations on bits (binary values).

    Operator Description Example
    & Bitwise AND a & b
    | Bitwise OR a | b
    ^ Bitwise XOR a ^ b
    ~ Bitwise Complement ~a
    << Left shift a << 2
    >> Right shift a >> 2

    üìù Explanation: - a & b: Bit is 1 only if both bits are 1. - a | b: Bit is 1 if either bit is 1. - a ^ b: Bit is 1 if bits are different. - ~a: Inverts (flips) all bits. - a << n: Shifts bits left by n (multiplies by 2‚Åø). - a >> n: Shifts bits right by n (divides by 2‚Åø).

    üî∏ Precedence and Associativity

    Operator precedence defines the order in which operators are evaluated.

    ‚úÖ Higher precedence operators are evaluated before lower precedence ones.

    üìå Example:

    int result = 10 + 2 * 5;  // result = 10 + (2 * 5) = 20

    ‚úÖ 4. Control Statements

    Control statements allow the program to make decisions and repeat actions.

    üî∏ if, else if, else

    Used for decision-making based on conditions.

    int age = 18;

if (age < 18) {
    System.out.println("Minor");
} else if (age == 18) {
    System.out.println("Just became adult");
} else {
    System.out.println("Adult");
}

    ‚úÖ The first matching condition is executed, others are skipped.

    üî∏ switch Statement

    Used to select one of many code blocks to be executed.

    int day = 2;

switch (day) {
    case 1: System.out.println("Sunday"); break;
    case 2: System.out.println("Monday"); break;
    default: System.out.println("Invalid day");
}

    ‚úÖ break prevents fall-through to next cases.

    üî∏ Loops (while, do-while, for)

    Used to repeat a block of code.

    üîπ while loop ‚Äì checks condition before execution:

    int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}

    üîπ do-while loop ‚Äì runs the code at least once, checks condition after:

    int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);

    üîπ for loop ‚Äì most common, compact form:

    for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

    üî∏ break, continue, return

    üîπ break ‚Äì exits the current loop or switch:

    for (int i = 1; i <= 5; i++) {
    if (i == 3) break;
    System.out.println(i);
}
// Output: 1 2

    üîπ continue ‚Äì skips the current iteration:

    for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;
    System.out.println(i);
}
// Output: 1 2 4 5

    üîπ return ‚Äì exits from the method completely:

    public void greet() {
    System.out.println("Hello");
    return;
    // Code here won‚Äôt execute
}

    ‚úÖ 5. Arrays

    Arrays store multiple values of the same type in a single variable.

    üî∏ 1D Arrays

    A linear collection of elements.

    int[] numbers = {10, 20, 30, 40};
System.out.println(numbers[0]); // Output: 10

    You can also initialize with size:

    int[] arr = new int[5];  // default values = 0
arr[0] = 100;

    üî∏ 2D Arrays

    An array of arrays (like a matrix or table):

    int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6}
};

System.out.println(matrix[1][2]);  // Output: 6

    üî∏ Array Traversal

    Use for or for-each loop:

    int[] arr = {10, 20, 30};
for (int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}

// For-each loop
for (int num : arr) {
    System.out.println(num);
}

    üî∏ Array vs ArrayList

    Feature Array ArrayList
    Fixed Size Yes No (can grow/shrink)
    Type Can store primitives Stores objects only (e.g. Integer)
    Performance Faster Slightly slower (dynamic resizing)
    Syntax int[] arr = new int[5]; ArrayList<Integer> list = new ArrayList<>();

    ### ‚úÖ 6. Methods
    Methods are blocks of code that perform specific tasks. They improve code reusability.
    ___
    #### üî∏ Defining and Calling Methods 
    public class Main {
    static void greet() {
        System.out.println("Hello!");
    }

    public static void main(String[] args) {
        greet();  // calling the method
    }
}

    üî∏ Method Overloading

    Multiple methods with same name but different parameters:

    void greet() {
    System.out.println("Hello");
}

void greet(String name) {
    System.out.println("Hello " + name);
}

    üî∏ return Keyword

    Used to return a value from a method:

    int add(int a, int b) {
    return a + b;
}

    üî∏ Pass-by-Value

    Java uses pass-by-value, meaning copies of variables are passed to methods:

    void changeValue(int x) {
    x = 50;
}

int a = 10;
changeValue(a);
// a is still 10 (not changed)

    For objects, the reference is passed by value, so internal fields can still be changed.

    void updateName(Person p) {
    p.name = "Alice";
}
